# Portfolio – Personal Developer Showcase

This is my personal developer portfolio built with Angular.  
It showcases my frontend development skills, selected projects, and technical background with a focus on modern UI/UX, performance, and clean architecture.

The application is designed as a responsive single-page portfolio that presents real-world projects in a structured and recruiter-friendly way.

## Features

- Modern responsive UI/UX design
- Multi-language support (German / English)
- Smooth scroll navigation between sections
- Component-based Angular architecture
- Project showcase with live demos and GitHub links
- Contact form with email integration
- Animations and interactive UI elements
- Clean and maintainable code structure

## Running Locally

### Prerequisites
- Node.js
- Angular CLI

### Setup

```bash
git clone git@github.com:A-Marbach/portfolio.git
cd portfolio
npm install
ng serve
```

## Application URL

```bash
http://localhost:4200
```
